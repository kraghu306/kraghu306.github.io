Microsoft Build 2015 Keynote

Raghuram Korukonda 6 May, 2015 03:33PM

If you haven't watched Microsoft's Build 2015. Here is a snapshot of something wonderful..!!

You can watch the Build 2015 keynote here: Build Keynote - By Satya Nadella

Here's a list of some interesting announcements from Build’s keynote:

Visual Studio Code: A free and light weight version of Visual Studio which now runs even in OSX and Ubuntu. That means you can write .NET code on your Mac and Ubuntu machines.
Link:Visual Studio Code

Microsoft launched Carrier-Billing for Windows Phone Store last year, now it is extended to all Windows 10 devices

Microsoft is now offering new APIs to integrate Third Party apps and Add-ins directly into complete Office Suite.

Two new products in Azure Family for developers:

  • SQL Data Warehouse(Easy to aggregate + store petabytes of data interactively query and visualize data Operationalize machine learning).

  • Azure Data Lake Service(A new data repository for big data analytics workloads).

Developing apps for Windows 10 can be donr in 4 wasy apart from natively developing for it:

  • Web: With new developer tools, Developers can register their Website with Windows Store to reuse server hosted website tools and rich capabilities of the universal Windows 10 platform like Cortana, Live Tile, Xbox Live, Push Notification and in-app purchases and turn them into powerful webapps, all from just a website.

  • .NET & Win32: Developers can now recompile their existing .NET & Win32 code and convert them to Universal apps, to put in store.

  • Java/ C++: Android apps made in Java and C++ can be easily re-compiled and moved to Windows store with almost no effort, along with current app functionality, they'll also gain Windows exclusive features like Live Tile and integration with Microsoft Services. They run in a Windows Security container that allow for a trustworthy experience.

  • Objective-C: Developers can now bring their iOS apps to Windows store in no time using the new tools.

Microsoft Edge: Microsoft Edge is the new super powerful browser from Microsoft replacing IE. Edge is deeply integrated with Cortana, OneNote and Reading list and the rich features like Reading Mode. EDGE supports third-party extensions. Now developers can port their web-extensions made via JavaScript and HTML from CHROME or FIREFOX to Edge with little or no effort.
Link: Microsoft Edge/Spartan

Continuum: How a PC with mouse input layout changes when switched to tablet mode. Now, Microsoft introduced CONTINUUM FOR PHONES. When users connect their phone to an external monitor via HDMI, along with a Bluetooth keyboard and mouse - it turns into a PC. User can access all the universal apps installed on their phone through the monitor with desktop like layout. In fact, user can use the phone simultaneously to communicate with others and still use the connected monitor like a Workstation PC.

Hololens: It's a device whose working must be watched in action.
Link:Hololens

Trial Azure Account with MSDN Benefits

Raghuram Korukonda 7 May, 2015 01:15PM

What is Azure ? In short, it’s Microsoft’s cloud platform: a growing collection of integrated services — Compute, Storage, Data, Networking, and Apps — that help you move faster, do more, and save money. You can build infrastructure, develop modern applications, gain insights from data, manage identity and access and more.

If you are an MSDN Subscriber you can create a Microsoft Azure free trial account, and you can activate the benefits available for MSDN subscribers.
For more details: Azure MSDN Benefits Details.

There are several ways to get an Azure subscription, such as the free trial subscription, the pay-as-you-go subscription, which has no minimums or commitments and you can cancel any time; Enterprise agreement subscriptions, or you can buy one from a Microsoft retailer. In this task you are going to create a free trial subscription.

  1. Browse to the following page http://azure.microsoft.com/en-us/pricing/free-trial/ to obtain a free trial account. Click the Try it Now button.

  2. Enter the credentials for the Microsoft account that you want to use. You will be redirected to the Sign up page.
  3. In the Mobile verification section, enter your mobile phone number, and click the Send text message button and when you receive the verification code, enter it in the corresponding box, and click Verify code.
  4. Fill in the Payment information tab(Add your credit card detials).And in the Agreement section, check the I agree to the subscription Agreement, offer details, and privacy statement option, and click Sign up

    Note: Your credit card will not be billed, unless you remove the spending limits. If you run out of credit, your services will be shut down unless you choose to be billed.

  5. Click the Portal link at the top-right of the page, to navigate to the Azure Management Portal. You will be greeted by the Windows Azure Tour dialog, which gives you a sense on how to move around in the Azure portal.
    Microsoft Azure tour

Activating MSDN benefits:
  1. If you happen to be an MSDN subscriber you can activate your Azure MSDN benefits. It is no charge, you can use your MSDN software in the cloud, and most importantly you get up to $150 in Azure credits every month. You can also get 33% discount in Virtual Machines and much more.
  2. To active the MSDN benefits, browse to the following URL: http://azure.microsoft.com/en-us/pricing/member-offers/msdn-benefits-details/
  3. Scroll down to see the full list of benefits you will get for being a MSDN member. There is even a FAQ section you can read.
  4. Click the Activate button to activate the benefits.
  5. You will need to enter your Microsoft account credentials to verify the subscription and complete the activation steps.
